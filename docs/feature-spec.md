# 기능 명세 (초안)

## 제품 한 줄

클라이언트·상품 정보를 입력하면 **이커머스/랜딩용 상세페이지 섹션**을 구조화해 생성·편집하는 웹 도구.

## 사용자

- 개인 외주자(1인) 본인 사용을 1차 대상으로 한다.

## 핵심 기능 (MVP 방향)

1. **프로젝트/클라이언트 단위**로 작업물을 구분한다.
2. **상품 스펙·USP·타깃·톤** 등 입력 폼으로 컨텍스트를 수집한다.
3. **섹션 템플릿**(히어로, 스펙, 비교, FAQ 등)을 선택해 **섹션별 카피**를 생성한다.
4. 생성문을 **인라인 편집**하고, 버전 또는 스냅샷 단위로 되돌릴 수 있게 한다(구현 단계에서 단순화 가능).
5. 결과를 **마크다운/HTML/클립보드** 등으로 내보낸다(포맷은 구현 시 확정).

## 비기능

- 비용: `docs/cost-saving-policy.md` 준수.
- 프라이버시: 클라이언트 민감 정보는 로컬 우선·불필요한 로그 금지.
- 속도: 섹션 단위 재생성이 가능해야 한다(전 페이지 매번 재생성 지양).

## 범위 밖 (초기)

- 결제·멀티 유저 SaaS 완전체
- 이미지 자동 합성(별도 도구 연동은 이후 검토)

## 오픈 이슈

- 데이터 저장: 로컬 전용 vs 클라우드 동기화
- 납품 포맷: 쿠팡/스마트스토어 등 마켓별 제약 반영 여부
